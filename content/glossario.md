# Glossário — Termos técnicos explicados

Lista completa com explicações curtas e diretas. Cada termo tem uma definição simples, sem jargão. Para aprofundar, vá ao tópico correspondente.

- **Airdrop** — Distribuição gratuita de tokens para usuários ativos de projetos cripto. Vira um "bônus" de fidelidade.
- **API** — Janela de atendimento entre programas. Um software pede, o outro entrega.
- **ASR** — Automatic Speech Recognition. IA que transforma áudio em texto (oposto de TTS).
- **Auth Bypass** — Falha onde um atacante consegue pular o sistema de autenticação sem credencial válida.
- **Authorization** — Autorização. Define **o que** um usuário pode fazer depois de autenticado.
- **Backend** — A parte do software que roda no servidor, invisível para o usuário. Lida com banco de dados, regras de negócio e APIs.
- **Bearer Token** — Formato padrão de autenticação via HTTP onde o token é enviado no header `Authorization`.
- **Blue Team** — Time responsável pela defesa em cibersegurança: monitoramento, detecção e resposta a incidentes.
- **Bug Bounty** — Programa onde empresas pagam hackers éticos por vulnerabilidades reportadas.
- **CDN** — Content Delivery Network. Rede de servidores que acelera carregamento de sites globalmente.
- **CI/CD** — Continuous Integration / Continuous Deployment. Deploy automático após os testes passarem.
- **CIA Triad** — Confidencialidade, Integridade, Disponibilidade. As três dimensões de qualquer falha de segurança.
- **Claude Code** — CLI oficial da Anthropic para usar Claude em tarefas de programação com skills, MCPs e agents.
- **CPM** — Cost Per Mille. Custo por mil impressões de um anúncio.
- **CTR** — Click-Through Rate. Porcentagem de quem viu um anúncio/link e clicou.
- **DNS** — Domain Name System. A "lista telefônica" da internet que traduz domínio em IP.
- **Docker** — Container que isola um programa e suas dependências para rodar em qualquer ambiente sem quebrar.
- **DOM** — Document Object Model. A estrutura de um site em memória que o JavaScript manipula.
- **E2E Encryption** — End-to-End Encryption. Criptografia onde só emissor e receptor conseguem ler — nem o servidor.
- **FFmpeg** — A ferramenta de processamento de vídeo/áudio mais poderosa do mundo. Open source.
- **Fine-tuning** — Treinar um modelo de IA existente com seus próprios dados para especializá-lo.
- **Frontend** — A parte visível do software: telas, botões, interações com o usuário.
- **GraphQL** — Alternativa ao REST onde o cliente pede exatamente os campos que precisa, em vez de receber tudo sempre.
- **GRC** — Governança, Risco e Conformidade. O "guarda-livros" da cibersegurança corporativa.
- **Hash** — Função matemática que transforma qualquer entrada num código de tamanho fixo. Usado para senhas (one-way).
- **Hook** — Script que roda automaticamente quando um evento acontece (ex: antes de um commit, depois de salvar).
- **IDOR** — Insecure Direct Object Reference. Trocar um ID na URL para acessar dados de outros usuários.
- **IDS / IPS** — Intrusion Detection System / Intrusion Prevention System. Detecta ou bloqueia ataques automaticamente.
- **JWT** — JSON Web Token. "Crachá digital" que prova quem você é sem consultar o banco a cada request.
- **Keyword** — Palavra-chave. Termo que alguém digita no Google — alvo central de SEO e de ads.
- **LGPD** — Lei Geral de Proteção de Dados (Brasil). Equivalente brasileiro do GDPR europeu.
- **LLM** — Large Language Model. Modelo de linguagem grande (GPT, Claude, Gemini). O "cérebro" dos chatbots.
- **MCP** — Model Context Protocol. Padrão para conectar IA a ferramentas, bancos e APIs externas.
- **MVP** — Minimum Viable Product. A versão mínima que você pode lançar para testar se vale a pena construir mais.
- **OHLCV** — Open, High, Low, Close, Volume. Dados históricos básicos de uma ação.
- **OSINT** — Open Source Intelligence. Coleta de informações públicas para investigar pessoas, empresas ou alvos.
- **Offramp** — Converter cripto em dinheiro fiduciário (dinheiro "real" — reais, dólares).
- **OWASP** — Open Web Application Security Project. Fundação referência em segurança web; publica o famoso "OWASP Top 10".
- **Payload** — O "conteúdo útil" de um ataque — a carga que de fato exploram uma falha (ex: o código XSS injetado).
- **Pentest** — Penetration Test. Teste de invasão autorizado para encontrar vulnerabilidades antes que criminosos encontrem.
- **Playwright** — Biblioteca de automação de browser (Chromium, Firefox, Safari) muito usada para testes e scraping.
- **Prompt Injection** — Ataque onde um texto manipula uma IA a fazer coisas que não deveria, sobrescrevendo suas regras.
- **PRD** — Product Requirements Document. Documento que define o que um produto precisa ter antes de começar a construir.
- **Purple Team** — Red Team + Blue Team trabalhando juntos (atacantes e defensores colaborando).
- **RAG** — Retrieval-Augmented Generation. IA que consulta seus documentos antes de responder, em vez de só usar conhecimento de treinamento.
- **Rate Limit** — Limite de quantas requisições um cliente pode fazer num período (ex: 100 por minuto).
- **RCE** — Remote Code Execution. A falha mais crítica da web: permite executar código arbitrário no servidor.
- **Red Team** — Time de atacantes éticos que simulam invasões reais para encontrar falhas.
- **REST** — Representational State Transfer. O estilo de API mais comum na web — endpoints fixos retornando JSON.
- **ROAS** — Return On Ad Spend. Retorno sobre o gasto em ads. ROAS 3x = $3 de receita para cada $1 gasto.
- **SaaS** — Software as a Service. Software que você paga mensalmente em vez de comprar uma licença fixa.
- **SEO** — Search Engine Optimization. Otimização para aparecer organicamente no Google.
- **Serverless** — "Sem servidor". Você escreve funções; a cloud gerencia os servidores. Paga só quando executa.
- **SIEM** — Security Information and Event Management. Ferramenta que coleta logs de toda a empresa para detectar comportamentos suspeitos.
- **Skill** — Em agentes de IA, um bloco de conhecimento reutilizável que ensina a IA a fazer uma tarefa específica.
- **SQL Injection (SQLi)** — Inserir comandos SQL através de campos de formulário para manipular o banco de dados.
- **SSL / TLS** — Protocolos que criam o "cadeado" de segurança em sites HTTPS. Criptografam dados em trânsito.
- **Steganografia** — Esconder dados dentro de arquivos (fotos, áudios, textos) de forma que ninguém perceba que há segredo.
- **Subdomínio** — Parte de um domínio antes do principal. Em `blog.exemplo.com`, `blog` é o subdomínio.
- **Token (IA)** — Unidade de medida de texto para IA. Uma palavra tem em média 1,3 tokens.
- **Token (Auth)** — Código que prova a identidade do usuário em requisições (ex: JWT).
- **TTS** — Text-to-Speech. IA que transforma texto em áudio falado.
- **URL** — Endereço de uma página na web. Ex: `https://exemplo.com/pagina`.
- **Vector DB** — Banco de dados especializado em busca semântica para IA — essencial para RAG.
- **VPN** — Virtual Private Network. Túnel criptografado que esconde seu IP e criptografa tráfego.
- **WAF** — Web Application Firewall. Firewall especializado em proteger aplicações web contra ataques.
- **Whisper** — Modelo de ASR da OpenAI que transcreve áudio em 50+ idiomas com alta precisão.
- **XSS** — Cross-Site Scripting. Injetar JavaScript malicioso em sites para atacar outros visitantes.
- **Zero-day** — Vulnerabilidade descoberta antes do fabricante saber que existe. Sem defesa conhecida.
