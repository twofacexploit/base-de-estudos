## O que é

**CloudScraper** é uma biblioteca Python que **simula um navegador humano** para passar pelas verificações anti-bot do Cloudflare. Quando você tenta fazer scraping direto, o Cloudflare bloqueia — o CloudScraper finge ser um browser real.

## Como funciona

Em vez de mandar uma requisição crua (que o Cloudflare detecta como "bot"), a biblioteca monta **headers HTTP** idênticos aos de um Chrome real: User-Agent, Accept-Language, Accept-Encoding, a ordem exata dos campos. Para o servidor, parece uma pessoa navegando.

## Diferença do CloudflareBypassForScraping (ver 1.23)

| Ferramenta | Como funciona | Peso | Confiabilidade |
|---|---|---|---|
| **CloudScraper** | Simula headers | Leve | Média |
| **CloudflareBypassForScraping** | Usa browser real (Playwright) | Pesado | Alta |

## Quando usar cada um

- **Sites simples** → CloudScraper (é rápido)
- **Sites com JS challenge** → o do 1.23 (Cloudflare evoluiu e exige execução de JavaScript)
