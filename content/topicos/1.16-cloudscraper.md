## O que é

**CloudScraper** é uma ferramenta para programadores que precisam coletar dados de sites de forma automática (scraping — como um robô que lê páginas da web). O problema é que muitos sites usam uma proteção chamada **Cloudflare** que bloqueia robôs. O CloudScraper ensina o robô a se disfarçar de pessoa humana para passar por essa proteção.

## Como funciona

Quando um programa tenta acessar um site diretamente, o Cloudflare percebe que não é um ser humano e bloqueia o acesso. O CloudScraper resolve isso montando uma "fantasia" — ele imita todos os detalhes de um navegador Chrome real: o idioma, a forma como pede os dados, a ordem exata das informações enviadas. Para o servidor, parece que uma pessoa está navegando normalmente.

## Diferença do CloudflareBypassForScraping (ver 1.23)

| Ferramenta | Como funciona | Peso | Confiabilidade |
|---|---|---|---|
| **CloudScraper** | Imita as informações de um navegador | Leve e rápido | Média |
| **CloudflareBypassForScraping** | Usa um navegador real por baixo dos panos | Pesado e mais lento | Alta |

## Quando usar cada um

- **Sites mais simples** → CloudScraper (é muito mais rápido)
- **Sites com desafios avançados de JavaScript** → o outro (o Cloudflare evoluiu e agora exige que o navegador execute programas JavaScript de verdade)

## Exemplos do dia a dia

1. Um pesquisador quer coletar automaticamente os preços de passagens aéreas de um site de viagens para comparar tendências ao longo do tempo. O site usa Cloudflare e bloqueia robôs. Com o CloudScraper, o programa consegue coletar os dados se passando por um usuário normal navegando no Chrome.

2. Um jornalista de dados quer monitorar diariamente um site de notícias governamental que usa proteção anti-bot. O CloudScraper permite que o script de coleta rode todo dia de manhã automaticamente, sem ser bloqueado, entregando os dados organizados para análise.
