## O que é

A Cloudflare lançou um endpoint oficial pra **crawling profissional**, usando a infraestrutura global dela.

```
POST https://api.cloudflare.com/client/v4/accounts/{id}/browser-rendering/crawl
```

## Características

- **Headless browser** — navegador invisível que executa JavaScript
- **Respeita robots.txt** por padrão (comportamento ético)
- **Crawl incremental** — pula páginas que não mudaram desde a última vez
- **Modo estático** — sem browser, só HTTP (mais rápido para sites simples)
- **Assíncrono** — você manda a URL, recebe um `job_id`, consulta o resultado depois

## Por que usar

- **Escala** — Cloudflare tem servidores no mundo inteiro, crawling massivo
- **Não bate bloqueios** — requisições saem da própria Cloudflare
- **Confiabilidade** — infraestrutura enterprise
- **Preço** — disponível no **plano gratuito** do Cloudflare

## vs Firecrawl (2.4)

Firecrawl é ótimo pra **uso geral** e começar rápido. O `/crawl` do Cloudflare é mais indicado quando você já usa Cloudflare na sua stack ou precisa de **escala massiva** — milhares de URLs em paralelo.
