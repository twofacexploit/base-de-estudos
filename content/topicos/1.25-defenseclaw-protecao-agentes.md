## O que é

O **DefenseClaw** (criado pela Cisco) é como um segurança que fica de olho em um assistente robótico dentro de uma empresa. Toda vez que o assistente de IA tenta fazer alguma coisa, o DefenseClaw verifica se aquela ação é segura antes de deixar acontecer.

## O problema que resolve

Assistentes de IA estão ganhando cada vez mais poderes: podem apagar arquivos, enviar mensagens, fazer compras online. O problema é que um comando errado ou uma instrução maliciosa pode fazer o assistente cometer um erro grave — como apagar dados importantes ou enviar informações confidenciais para o lugar errado.

## Classificação de ações

O DefenseClaw coloca cada ação em uma de três categorias:

- **SEGURO:** ler um documento, pesquisar na internet, consultar informações públicas
- **SUSPEITO:** acessar pasta fora do projeto, criar arquivos em locais incomuns
- **PERIGOSO:** apagar muitos arquivos de uma vez, enviar dados para servidor externo, executar comandos com poderes de administrador

Ações **perigosas** são bloqueadas automaticamente — só passam com a aprovação de um humano.

## Especialmente importante em

- **Empresas** onde um erro pode causar prejuízo financeiro grande
- **Hospitais e usinas** onde um erro pode afetar vidas
- **Ambientes de produção** — sistemas que estão funcionando de verdade, com dados reais

## A analogia

É como o limite do cartão de crédito: o banco não deixa você gastar acima de um valor sem confirmação extra. O DefenseClaw faz o mesmo com o assistente de IA — ações de alto impacto precisam de uma confirmação humana.

## Exemplos do dia a dia

1. Uma empresa usa um assistente de IA para organizar documentos. Quando o assistente tenta apagar uma pasta inteira por engano, o DefenseClaw bloqueia a ação e pede confirmação humana — como uma mensagem de "tem certeza que quer apagar?" antes de esvaziar a lixeira.

2. Um hospital usa um agente de IA para agendar consultas. Se o agente recebe uma instrução suspeita para enviar dados de pacientes para um endereço desconhecido, o DefenseClaw bloqueia imediatamente e alerta o administrador.
