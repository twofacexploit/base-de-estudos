## O que é

**LACP** (Local Agent Control Plane) é um **dashboard** que unifica o controle de múltiplos agentes de IA. Em vez de ter Claude Code em uma aba, Codex em outra, Cursor em outra, você tem **um painel central** vendo tudo ao mesmo tempo.

## O que oferece

- **Logs unificados** — o que cada agente fez, quando, em qual projeto
- **Métricas** — consumo de tokens, tempo de resposta, taxa de sucesso
- **Configuração** — mudar regras e permissões de todos os agentes num único lugar
- **Orquestração** — delegar tarefas entre agentes diferentes conforme o melhor pra cada caso

## Por que isso vira necessário

Quem trabalha com IA intensivamente rapidamente acumula **vários agentes**: um pra código, outro pra escrita, outro pra pesquisa. Sem centralização, vira **caos** — você não sabe mais o que cada um está fazendo, quanto está gastando, onde há sobreposição.

O LACP é a **sala de controle** pra isso — como um cockpit onde você vê todos os instrumentos de uma vez.

## Comparação útil

- **Claude Code** — um agente poderoso pra código
- **oh-my-claudecode** (2.21) — orquestração dentro do Claude Code
- **LACP** — nível acima: orquestração **entre** agentes diferentes (Claude + Codex + Cursor)

Cada camada resolve um tipo diferente de complexidade.
