## O que é

**LACP** é um painel central — pense numa central de monitoramento como a dos aeroportos, onde uma pessoa vê todas as telas de uma vez. Em vez de abrir cada programa de IA em uma aba diferente, você tem **um único lugar** onde acompanha tudo ao mesmo tempo.

## O que oferece

- **Logs unificados** — um histórico de tudo que cada IA fez, quando fez e em qual projeto, como um diário de bordo
- **Métricas** — quantos créditos cada IA consumiu, quanto tempo levou, quantas tarefas completou com sucesso
- **Configuração** — muda as regras de todas as IAs de uma vez só, como ajustar o volume da TV pela TV a cabo em vez de ir em cada aparelho
- **Orquestração** — direciona cada tarefa para a IA mais adequada, como um gerente que distribui o trabalho certo para o funcionário certo

## Por que isso vira necessário

Quem usa IA com frequência logo acaba tendo várias: uma para escrever textos, outra para fazer imagens, outra para organizar dados. Sem um painel central, vira bagunça — você não sabe quanto está gastando, o que cada uma fez, se estão trabalhando em coisas repetidas.

O LACP é como a sala de controle de uma fábrica — você vê todas as máquinas funcionando de uma vez, e age rápido se algo der errado.

## Comparação útil

- **Claude Code** — uma IA poderosa para criar códigos
- **oh-my-claudecode** (2.21) — coordena vários assistentes dentro do Claude Code
- **LACP** — nível acima: coordena IAs completamente diferentes entre si (Claude + outras IAs)

Cada camada resolve um tipo diferente de complexidade.

## Exemplos do dia a dia

1. Um gerente de marketing usa três IAs diferentes: uma para criar textos de anúncio, outra para analisar os resultados e outra para sugerir imagens. Com o LACP, ele vê tudo em uma tela só — quanto cada uma custou, o que cada uma fez, e pode dar novas ordens para todas de uma vez.

2. Uma pequena agência tem funcionários usando IAs diferentes para clientes diferentes. O LACP mostra para o dono quais IAs estão sendo usadas, por quem, em qual projeto — como um painel de controle do negócio em tempo real.
