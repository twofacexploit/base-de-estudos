## O que é

Ferramenta que usa **Playwright** (automação de browser real) pra resolver **JavaScript challenges** do Cloudflare. Obtém o cookie de validação e reusa em requisições seguintes.

## Como funciona

1. Abre um **Chromium real** em modo headless
2. Navega até a URL bloqueada pelo Cloudflare
3. O Cloudflare manda um desafio de JavaScript (calcular um hash, resolver captcha invisível)
4. O browser real executa o JS naturalmente, como um usuário humano faria
5. Cloudflare libera → ferramenta captura o cookie de sessão
6. Você reusa esse cookie em requisições Python normais, bem mais leves

## vs CloudScraper (1.16)

| Ferramenta | Abordagem | Peso |
|---|---|---|
| **CloudScraper** | Simula só headers HTTP | Leve, rápido |
| **CloudflareBypassForScraping** | Roda browser de verdade | Pesado, confiável |

## Quando usar

O Cloudflare evoluiu — hoje a maior parte dos sites exige **execução de JS**, o que só um browser real resolve. Então, na prática, o 1.23 é o que funciona na maioria dos casos modernos. O 1.16 só serve pra sites com proteção mais antiga.
