## Como funciona — 4 passos

1. Você tira um **screenshot** do site original (ou aponta a URL)
2. Uma **IA de visão** analisa: layout, cores, fontes, espaçamentos, proporções
3. Gera **HTML + CSS equivalente** do zero
4. Resultado: **pixel-perfect** — olhando, parece idêntico

## Usos legítimos

- **Aprendizado** — "como o Stripe faz aquele hero?" → clona, estuda, adapta
- **Prototipagem** — começar com uma estrutura que funciona, customizar depois
- **Análise competitiva** de design — ver o código por trás de um padrão visual
- **Inspiração** — quando você sabe o que quer mas não sabe implementar

## O que NÃO fazer

- **Copiar identidade visual** de concorrentes (problema legal e ético)
- **Clonar e colocar no ar** como se fosse seu (violação de direitos)
- **Roubar fontes, imagens, copy** — o clonador copia tudo, mas não tem direito de uso

## O diferencial vs ferramentas antigas

Antes, ferramentas de "clonar site" baixavam HTML cru — que vinha com toda a bagunça de frameworks, minificação, classes aleatórias. **Ilegível.**

IA de visão gera **código limpo do zero**, semântico, organizado — como se um humano tivesse escrito olhando pro design. É muito mais útil pra aprender e adaptar.

## Ferramentas do gênero

- **v0.dev** (da Vercel) — gera componentes React a partir de descrições
- **Screenshot-to-code** — versão open source
- **Locofy** — foca em converter para código de produção
