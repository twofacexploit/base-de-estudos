## A regra de ouro

**Não existe "a melhor linguagem".** Existe a linguagem **certa** para o objetivo que você tem. Essa é a primeira coisa que qualquer iniciante precisa aceitar — senão fica eternamente pulando de uma pra outra sem dominar nenhuma.

## Web Development

As linguagens que dominam a construção de sites e aplicações web:

**HTML, CSS, JavaScript, TypeScript, PHP, React JS, Angular, Python, Java, SQL**

HTML/CSS/JS é a base — todo site no mundo usa os três. React é o framework mais popular hoje. TypeScript vem ganhando força rapidamente.

## Software Development

Para programas desktop, sistemas, apps mobile e tudo que não é necessariamente web:

**C, C++, Java, Python, C#, Swift, Kotlin, Rust, Go, Ruby**

C e C++ ainda dominam onde performance importa (jogos, sistemas operacionais). Swift é Apple, Kotlin é Android. Rust é o futuro em muitos cenários.

## Machine Learning

O reino do Python:

**Python (rainha), R, Julia, Java, C++, Scala, MATLAB, JavaScript, Lisp, Prolog**

Python domina porque tem as bibliotecas mais maduras (NumPy, Pandas, TensorFlow, PyTorch). R é forte em estatística acadêmica. Julia é o "novo Python" em nicho científico.

## O destaque: Python

**Python é a linguagem mais versátil hoje.** Aparece nas **3 categorias** acima — web, software e IA. Se você quer uma única linguagem que sirva pra muita coisa, é Python.

## Por onde começar?

Depende do seu objetivo:

### Quer criar sites?
```
HTML → CSS → JavaScript → React
```
Sequência clássica. Cada um constrói sobre o anterior.

### Quer trabalhar com IA?
```
Python → NumPy → Pandas → TensorFlow/PyTorch
```
Python é indispensável. Depois, aprenda as bibliotecas na ordem.

### Quer cibersegurança?
```
Python + C + Bash
```
Python pra automação, C pra entender exploits, Bash pra scripts de automação em Linux.

### Quer apps mobile?
- **Android** → Kotlin
- **iOS** → Swift
- **Ambos ao mesmo tempo** → React Native ou Flutter

## A pior pergunta

**"Qual linguagem devo aprender primeiro?"**

A pior pergunta porque não tem resposta sem saber o que você quer fazer. A pergunta certa é: **"Qual problema eu quero resolver?"** — daí a linguagem surge naturalmente.
