## O que mudou

O **Claude Code Desktop** agora tem um **MCP built-in** que abre um preview visual do site conforme o Claude vai construindo — na mesma tela, sem você precisar abrir o navegador manualmente.

## Dev Server

Um **dev server** é um servidor local que roda o site em `localhost` durante o desenvolvimento. Toda vez que você salva um arquivo, ele recompila e atualiza o browser sozinho.

Antes: você tinha que abrir o terminal, rodar `npm run dev`, abrir o navegador, refrescar. Agora: Claude faz **tudo sozinho**.

## Resultado prático

**Zero passos manuais** — Claude cria o código, sobe o servidor, mostra o resultado visual ao lado do código. Você fala "muda o botão pra azul" e vê a mudança na mesma hora.

## Por que isso é um salto enorme

O maior atrito pra quem faz frontend com IA era: "pedi uma mudança → preciso ir ver se funcionou → achar o erro → voltar pro Claude → explicar o erro". Com preview nativo, esse ciclo encurta de **minutos para segundos**.

- **Frontend Design Skill** — Claude já pensando em design system
