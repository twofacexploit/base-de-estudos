## O stack completo

| Ferramenta | Função | Custo |
|---|---|---|
| **Claude** | Coding partner | $20/mês |
| **Supabase** | PostgreSQL + Auth + Storage | Gratuito |
| **Vercel** | Deploy + CDN global | Gratuito |
| **Namecheap** | Domínio profissional | ~$12/ano (~$1/mês) |
| **Stripe** | Pagamentos | 2.9%/transação (só paga quando vende) |
| **GitHub** | Versionamento + CI/CD | Gratuito |
| **Resend** | Emails transacionais (3k/mês) | Gratuito |
| **Clerk** | Autenticação (social, 2FA, magic links) | Gratuito |
| **Cloudflare** | DNS + CDN + DDoS | Gratuito |
| **PostHog** | Analytics + session replay | Gratuito |
| **Sentry** | Error tracking em produção | Gratuito |
| **Upstash** | Redis serverless (cache, filas) | Gratuito |
| **Pinecone** | Vector DB para IA e RAG | Gratuito |

**Total: ~$21/mês** até você ter receita real.

## Por que esses são os "certos"

Cada ferramenta dessa lista tem duas características essenciais:

1. **Free tier generoso** — cobre meses de uso antes de cobrar
2. **Escala suave** — quando você passa do free, o preço é proporcional (não um salto brutal)

Você não fica preso ao produto quando cresce — o custo acompanha sua receita.

## A filosofia

**Comece com o que é gratuito.** Valide o produto. Só quando tiver receita recorrente, mude para o plano pago. A maior parte dos projetos morre **antes** de precisar pagar por qualquer coisa — então gastar $500/mês em infra antes de validar o negócio é puro desperdício.

## O que muda quando você escala

Quando seu SaaS começa a crescer, o primeiro custo real costuma ser:

- **Vercel** pra tráfego alto (~$20/mês no Pro)
- **Supabase** quando passa do free tier (~$25/mês)
- **Stripe** — é % fixa, proporcional às vendas
- **Email transacional** volume alto (~$20-50/mês)

Mesmo em escala razoável, **não passa de $200/mês** por um bom tempo — e a essa altura você já deve estar com alguma receita significativa.

## A grande lição

SaaS **não é caro de começar**. É caro **mal planejado**. Começar com o stack certo economiza milhares de dólares e meses de retrabalho.
