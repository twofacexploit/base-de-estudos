## O que faz

**Glance** dá ao Claude Code a capacidade de **abrir sites**, **ver interfaces visualmente** (screenshots) e **interagir** como humano (clicar, digitar, rolar).

## Casos de uso

- **Testes automatizados** de fluxos completos ("abre o site, faça login, adicione item no carrinho")
- **Web scraping** de sites que bloqueiam scrapers tradicionais (porque é browser real)
- **Automação** de tarefas repetitivas na web (downloads, formulários, coleta de dados)
- **Monitoramento** de preços e mudanças de conteúdo

## Por que é diferente de outras automações

Ferramentas como Selenium e Playwright já automatizavam browser — mas precisavam de **código bem específico** ("clique no elemento `#submit-btn`"). Quebravam toda vez que o site mudava.

Glance usa **visão** da IA. Você fala: *"clique no botão azul de 'comprar'"* — e ela encontra visualmente o elemento, sem depender de seletores que quebram.

## A combinação poderosa

**Claude Code + Glance + Tavily + Firecrawl** = um agente que:

1. Busca informações na web (Tavily)
2. Extrai conteúdo de páginas (Firecrawl)
3. Interage com sites (Glance)
4. Analisa e decide (Claude Code)

É o que a maioria das pessoas **imagina** quando ouve "assistente de IA" — mas quase ninguém sabe que já existe.
