## O que é um repositório — e por que isso importa

Antes de falar dos 9, uma explicação simples sobre o que é um "repositório" (chamado carinhosamente de "repo" no mundo da tecnologia).

Pense numa biblioteca pública. Ela tem livros organizados em prateleiras, catalogados por assunto, disponíveis para qualquer pessoa pegar emprestado gratuitamente. Um repositório de tecnologia é a mesma coisa, mas para programas, ferramentas e configurações. É um endereço na internet onde alguém guardou algo útil que criou, organizou e disponibilizou para outros usarem gratuitamente.

O GitHub é o maior repositório de repositórios do mundo — é onde a maioria desses programas e ferramentas ficam armazenados. Quando alguém fala em "estrelas no GitHub", está falando de aprovações — como as avaliações de uma loja no Google: quanto mais estrelas, mais pessoas aprovaram.

Os 9 repositórios listados aqui foram selecionados especificamente por multiplicar a velocidade e qualidade de quem trabalha com o Claude Code — alguns já vistos em outros tópicos, outros novos. A ideia do "10x" no título é que essas ferramentas podem multiplicar por 10 a velocidade e qualidade do que você produz.

## Por que isso importa para você?

Existe uma diferença enorme entre alguém que descobre o Claude Code e sai usando no modo padrão, e alguém que, além do Claude, instala os recursos certos ao redor dele. É como a diferença entre trabalhar numa mesa vazia com só um lápis, e trabalhar numa mesa bem equipada com todos os materiais organizados à mão — régua, borracha, marcadores de cores, cola, tesoura, tudo no lugar certo.

Esses 9 repositórios são o equipamento que transforma a mesa vazia numa estação de trabalho completa. E o melhor: tudo gratuito, tudo testado pela comunidade, tudo disponível agora.

## Claude-Mem — a memória que a IA não tem nativamente

Já explicado em detalhe no tópico 2.9, o Claude-Mem resolve o problema mais frustrante de usar IA: ela esquece tudo quando você fecha e abre de novo. Com o Claude-Mem, o contexto do projeto é salvo automaticamente e carregado na próxima sessão.

O valor de listar aqui: se você vai instalar só uma coisa da lista, que seja esta. O impacto é imediato, universal (funciona em qualquer tipo de projeto) e elimina um atrito diário que você provavelmente já sofreu várias vezes sem saber que tinha solução.

## UI UX Pro Max — o designer que você não precisa contratar

Já explicado em detalhe no tópico 2.12. Com 42.000 downloads mensais e 43.000 estrelas, é a ferramenta de design mais popular do ecossistema Claude. Se você cria qualquer coisa visual — sites, aplicativos, painéis, apresentações — esta é indispensável.

## n8n-MCP — automação para humanos

O n8n é uma plataforma de automação que conecta mais de 400 serviços diferentes sem precisar programar. O "n8n-MCP" é a integração que conecta o Claude Code ao n8n — permitindo que você peça em linguagem normal para a IA criar automações.

Em vez de ter que aprender a configurar o n8n visualmente (já relativamente simples, mas ainda assim um processo de aprendizado), você simplesmente diz para a IA o que quer automatizar. Ela cria o fluxo no n8n por você. É como explicar para um secretário o que você quer que aconteça automaticamente, e ele configura o sistema sem você precisar aprender a usar o sistema.

## Obsidian Skills — sua anotações, mais inteligentes

Obsidian é um aplicativo popular de anotações pessoais. Muitas pessoas o usam para organizar pesquisas, projetos, ideias e estudos numa estrutura interconectada — como um segundo cérebro digital. O Obsidian Skills é uma coleção de habilidades que conecta o Claude Code ao vault (o arquivo de notas) do Obsidian.

Com essa conexão, quando você trabalha num projeto com a IA, ela pode buscar nas suas próprias anotações para encontrar informações relevantes. Se você anotou meses atrás os resultados de uma pesquisa sobre seu mercado, a IA pode encontrar e usar essa informação sem você precisar copiar e colar nada. É como ter um assistente que não só sabe o que você precisa agora, mas também leu todos os seus cadernos de anotações e pode consultar qualquer informação que você guardou lá.

## LightRAG — busca que entende significado

"RAG" significa "Retrieval-Augmented Generation" — uma técnica para buscar informações em documentos e usar esses resultados para alimentar a IA. Em linguagem simples: é um sistema que permite que a IA consulte uma coleção de documentos seus quando precisar de informação.

A palavra "Light" no nome indica que é uma versão mais eficiente e acessível do que outras soluções similares. E a principal diferença do LightRAG para uma busca simples é que ele entende o **significado** das suas perguntas, não só as palavras exatas.

**Busca simples:** Você pergunta "banco" → o sistema procura documentos que contêm a palavra "banco" → acha documentos sobre banco de dados, banco de empregos, banco de sangue, banco financeiro — misturado.

**LightRAG:** Você pergunta "onde guardar dinheiro com segurança" → o sistema entende que você está falando de finanças → acha documentos sobre poupança, tesouro direto, fundos, conta corrente — tudo relevante para o contexto, mesmo sem usar a palavra exata que você digitou.

Para quem trabalha com grande volume de documentos — relatórios, contratos, pesquisas, artigos — essa diferença é enorme. A busca passa a funcionar como um especialista humano que entende o que você quer, não como um robô que busca palavras literais.

## Everything Claude Code — o guia que todos deveriam ler

É uma coleção organizada das melhores configurações, dicas, truques e padrões para usar o Claude Code de forma profissional. Não é um programa — é documentação e exemplos práticos.

Pense como o manual de instruções completo que ninguém nunca lê, mas que contém a solução para a maioria dos problemas que as pessoas enfrentam. A diferença é que este vale a pena ler — especialmente a seção inicial, que economiza horas de descoberta por tentativa e erro.

## Superpowers — tudo de uma vez

O Superpowers é um pacote que junta mais de 20 habilidades aprovadas e testadas pela comunidade num único conjunto. Em vez de pesquisar e instalar habilidades individualmente — o que pode levar tempo — o Superpowers entrega um conjunto coerente de capacidades já integradas.

Com 96.000 estrelas no GitHub, é um dos projetos mais aprovados de todo o ecossistema. Quando você instala o Superpowers, está instalando o conjunto de ferramentas que a comunidade considerou mais valioso — como receber uma recomendação de um amigo de confiança em vez de ter que fazer toda a pesquisa sozinho.

## Awesome Claude Code — o guia vivo da comunidade

"Awesome" é uma tradição do GitHub: existe uma lista chamada "Awesome [nome da tecnologia]" para as principais ferramentas, e ela é mantida pela comunidade com os melhores recursos de cada área. O Awesome Claude Code é essa lista para o ecossistema Claude.

Funciona como um cardápio sempre atualizado: quando você quer descobrir se existe algo para uma necessidade específica, é o primeiro lugar para olhar. A comunidade mantém a lista atualizada, remove o que ficou obsoleto e adiciona o que foi lançado recentemente.

## GSD (Get Shit Done) — o gerente de foco

"Get Shit Done" significa literalmente "fazer as coisas acontecerem" — é um repositório que ensina a IA a manter o foco absoluto numa tarefa até terminá-la, sem se distrair com detalhes secundários, sem desviar do objetivo, sem fazer mais do que foi pedido.

Um problema comum ao trabalhar com IA é que ela às vezes "divaga" — começa a fazer uma coisa, vê algo adjacente interessante, adiciona funcionalidades que não foram pedidas, ou fica tentando fazer tudo de uma vez em vez de focar no que foi pedido. O GSD resolve isso: coloca a IA no modo "cabeça baixa, execute a tarefa, entregue o resultado, pare."

Para projetos onde a disciplina de execução é crítica — entrega de software, prazos apertados, tarefas bem definidas que precisam ser feitas corretamente antes de avançar — o GSD é a ferramenta certa.

## Por onde começar — a ordem que faz sentido

Se você está começando e não sabe o que instalar primeiro, siga essa ordem baseada em impacto imediato:

1. **Claude-Mem** primeiro — resolve o problema mais universal e frustrante imediatamente
2. **UI UX Pro Max** segundo — se você cria qualquer coisa visual, o retorno é instantâneo
3. **Everything Claude Code** — leia antes de continuar a instalar mais coisas, economiza horas
4. **Superpowers** — instala um conjunto sólido de capacidades de uma vez
5. O resto conforme a necessidade surgir — não tente instalar tudo de uma vez

## Exemplos do dia a dia

1. Um escritor que usa o Obsidian há dois anos para organizar sua pesquisa decide finalmente usar a IA para acelerar a escrita de um livro. Com o Obsidian Skills instalado, a IA tem acesso a todos os anos de notas de pesquisa, personagens desenvolvidos, rascunhos de cenas e referências que ele acumulou. Quando ele pede para a IA ajudar a desenvolver um capítulo, ela busca nas anotações o que é relevante para aquele momento da história — como ter um assistente que leu todos os seus cadernos e pode citar exatamente a pesquisa que você fez em 2023 sobre o período histórico que está descrevendo. O que antes era um processo de abrir a nota certa, ler, copiar o trecho relevante e colar agora é automático — a IA já sabe o que procurar e onde encontrar.

2. Uma equipe de design de produtos usa o GSD para garantir que quando pede para a IA criar um componente específico — um botão, um formulário, um menu — ela entrega exatamente aquilo, sem adicionar funcionalidades não pedidas, sem modificar outras partes do código, sem "sugerir melhorias" enquanto trabalha. O processo de desenvolvimento fica muito mais previsível: cada tarefa é clara, a IA executa sem desviar, o resultado é exatamente o esperado. É como um marceneiro que segue o projeto exatamente como pedido — sem adicionar detalhes que não foram aprovados ou modificar outras partes da casa sem avisar.
