## O autor

**Souhaib Naceri** (h4x0r_dz) — pesquisador argelino conhecido por encadear bugs pequenos em impactos gigantes.

## A lição principal

Bugs isolados costumam pagar pouco. **Bugs encadeados** — onde um prepara o terreno para o próximo — pagam muito. Este case mostra exatamente isso.

## As 3 vulnerabilidades encadeadas

### 1. JWT Realm Manipulation — $3.000

```
Login normal:  {"realm": "test-user"}
Ataque:        {"realm": "test-dashboard"}
Resultado:     Acesso ao painel admin
```

O servidor aceitava o campo `realm` sem validar qual valor era permitido.

### 2. Auth Bypass removendo "Bearer" — $20.000 (compartilhado)

```
Com Bearer:    Authorization: Bearer TOKEN → 401 ❌
Sem Bearer:    Authorization: TOKEN        → 200 ✅
```

Um middleware rejeitava o prefixo "Bearer" mas deixava passar o token cru.

### 3. Arbitrary File Overwrite

Ele descobriu que upload de arquivo ia direto pro bucket S3 da Amazon da empresa — **sem validação do caminho de destino**. Podia sobrescrever:

- JavaScript do site principal
- Arquivos `.exe` que usuários baixavam
- PDFs oficiais

Resultado teórico: **RCE (Remote Code Execution)** nos computadores de todos os usuários do site.

## Ferramentas usadas

- **Burp Suite** — interceptar e modificar requests
- **ffuf** — fuzzing de endpoints
- **Wayback Machine** — achar endpoints antigos removidos
- **JWT.io** — decodificar tokens JWT

## A plataforma

Relatório submetido via **HackerOne**. Pagamento total: **$23.000**.

## A moral

Nenhum desses bugs sozinho seria crítico. Juntos, viraram um cenário de catástrofe — e o caçador foi pago de acordo.
