## O que faz

Pega **qualquer URL** e retorna o conteúdo limpo, estruturado, pronto pra ser consumido por uma IA. Markdown, HTML ou JSON — você escolhe.

## Por que é diferente de copiar HTML

HTML cru de um site moderno é **caótico** — scripts, anúncios, menus, banners. A IA se perde lendo tudo isso. Firecrawl:

- **Remove**: anúncios, navbars, cookies banners, scripts, trackers
- **Mantém**: conteúdo útil e estruturado
- **Suporta**: JavaScript rendering (para Single Page Apps modernos)

## Casos de uso

- **RAG** — criar base de conhecimento a partir de qualquer site
- **Scraping** — sem precisar lidar com HTML bagunçado manualmente
- **Treinamento** — converter centenas de páginas em dataset limpo
- **Monitoramento** — comparar versões ao longo do tempo

## SDKs disponíveis

- **Python**
- **Node.js**

## Integrações nativas

- **LangChain** (framework mais popular de agentes)
- **LlamaIndex** (framework de RAG)
- **CrewAI** (multi-agente)
- **Dify** (plataforma visual de agentes)

## A economia de tempo

Antes: escrever um scraper custom pra cada site, com regex, BeautifulSoup, Playwright. **Horas por site.** Agora: uma linha de código, qualquer site, **segundos**.
