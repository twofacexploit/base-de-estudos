## O que é

**Sherlock** busca um nome de usuário em **400+ redes sociais** ao mesmo tempo. Em segundos, você sabe em que sites aquele username existe — com links diretos.

## Como usar

```bash
sherlock username              # busca básica
sherlock username --output resultado.txt
sherlock user1 user2 user3    # múltiplos
sherlock username --csv       # exporta CSV
sherlock username --dry-run   # preview sem executar
```

## O que retorna

Lista de todas as redes onde aquele username está registrado, com URLs. Se você usa o mesmo nickname no Twitter, GitHub, Instagram, LinkedIn e forums obscuros — o Sherlock acha tudo.

## Lições de privacidade

Isso ensina uma coisa **assustadora**: reusar username é um vazamento de privacidade **enorme**. Alguém que descobre seu @ do Twitter pode mapear sua vida inteira online em 30 segundos.

**A contramedida:** use usernames diferentes em contextos diferentes (profissional, pessoal, anônimo). Se quer privacidade máxima, **nunca reuse**.
