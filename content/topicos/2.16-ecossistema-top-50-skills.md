## Skills oficiais Anthropic

Skills mantidas pela própria Anthropic — atualizadas, confiáveis, bem documentadas:

- **PDF, DOCX, PPTX, XLSX** — manipulação de documentos
- **Doc Co-Authoring** — escrever textos colaborativamente
- **Frontend Design** — 277.000 instalações
- **Canvas Design, Algorithmic Art, Theme Factory**
- **Web Artifacts Builder**
- **Superpowers** — 96.000 stars
- **Skill Creator** — criar suas próprias skills

## MCP Servers essenciais

MCP = Model Context Protocol, o "USB-C" dos agentes de IA.

- **Tavily** — busca estruturada pra IA (dados limpos, não links azuis)
- **Context7** — documentação atualizada injetada no contexto (`"use context7"` no prompt)
- **Task Master AI** — gerente de projetos: PRD → tarefas → Claude executa

## Frameworks de agentes

- **AutoGPT**
- **LangGraph** (LangChain + grafos de estado)
- **CrewAI** (multi-agente colaborativo)
- **Dify** (plataforma visual)
- **OWL**
- **CopilotKit**

## IA local

Pra quem não quer depender de nuvem:

- **Ollama** — runtime de LLMs locais
- **Open WebUI** — interface tipo ChatGPT para Ollama
- **LlamaFile** — modelo + runtime em um arquivo só
- **Unsloth** — fine-tuning rápido de LLMs

## Automação

- **n8n** — 400+ integrações, alternativa open source ao Zapier
- **Langflow** — drag-drop visual para agentes

## Catálogos de skills

- **skillsmp.com** — 80.000+ skills
- **aitmpl.com/skills** — templates prontos
- **skillhub.club** — 31.000+ skills com rating

## O overwhelm positivo

Parece muita coisa — porque é. Mas **ninguém precisa aprender tudo**. Escolha 3-5 ferramentas do seu contexto (o que você faz no dia a dia) e mergulhe nelas. O resto fica pra quando fizer sentido.
