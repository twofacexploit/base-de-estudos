## O que é Red Team

Profissionais contratados para **atacar os próprios sistemas da empresa**, encontrando falhas antes que criminosos reais o façam. Pense neles como "bombeiros que ateiam fogo no prédio pra ver se os detectores funcionam".

## As 8 ferramentas clássicas

| Ferramenta | O que faz | Analogia |
|---|---|---|
| **Nmap** | Escaneia portas e serviços de um alvo | Bate em todas as portas de uma casa |
| **Burp Suite** | Intercepta tráfego entre browser e site | Grampo telefônico para web |
| **Metasploit** | Caixa de ferramentas com exploits prontos | Kit de ladrão profissional |
| **Wireshark** | Captura e lê pacotes de dados da rede | Escuta toda conversa da rede |
| **SQLmap** | Testa injeção de SQL automaticamente | Testa todas as senhas de cofres |
| **Hydra** | Tenta senhas automaticamente (brute force) | Testa 10.000 chaves por segundo |
| **John the Ripper** | Quebra hashes de senhas | Decifra senhas embaralhadas |
| **Aircrack-ng** | Testa segurança de redes Wi-Fi | Testa se o cadeado do Wi-Fi é fraco |

## Por que isso importa

Cada uma dessas ferramentas testa uma "camada" do sistema. Juntas, elas formam o arsenal básico de qualquer pentester profissional. A maioria é **gratuita e open source** — o que separa o profissional do amador é saber combiná-las.
