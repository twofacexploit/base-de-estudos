## O que é

Uma skill que permite controlar o **FFmpeg** (a ferramenta de processamento de vídeo mais poderosa do mundo) via **linguagem natural** dentro do Claude Code. Você fala o que quer, Claude traduz em comandos FFmpeg.

## Capacidades

- **Cortar, juntar e dividir** vídeos
- **Adicionar e extrair áudio**
- **Gerar legendas** automaticamente (usando Whisper)
- **Filtros e efeitos** visuais
- **Exportar otimizado** para YouTube, Instagram, TikTok (cada plataforma tem seu preset)
- **Criar thumbnails** automaticamente a partir de frames específicos

## Antes e depois

**Antes** (editando no Premiere/DaVinci):
```
Abrir o software → Importar → Arrastar na timeline  
→ Achar o ponto certo → Cortar → Adicionar legenda  
→ Escolher fonte → Exportar na resolução certa  
→ (horas por vídeo)
```

**Agora**:
```
"Corta os primeiros 30 segundos, adiciona legenda  
automática e exporta para TikTok 9:16"
→ feito em minutos
```

## Por que FFmpeg é poderoso

FFmpeg é **open source**, rodando em quase todo servidor do mundo — Netflix, YouTube, Twitch usam internamente. O problema é que **a sintaxe é horrível**:

```bash
ffmpeg -i input.mp4 -vf "scale=1080:1920,pad=1080:1920:(ow-iw)/2:(oh-ih)/2:black" 
       -c:v libx264 -preset fast -crf 23 output.mp4
```

A skill esconde tudo isso atrás de linguagem natural — você não precisa mais decorar flags.

## Quando usar

- **Criadores de conteúdo** que postam em múltiplas redes
- **Podcasters** que precisam gerar clipes curtos a partir de episódios longos
- **Empresas** que fazem dezenas de vídeos similares (onboarding, tutoriais)
- **Pós-produção em lote** — aplicar mesma edição a 50 vídeos

Horas de trabalho viram minutos. É uma mudança real de produtividade pra quem vive de vídeo.
