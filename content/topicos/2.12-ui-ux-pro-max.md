## O que é

Uma **skill** que transforma o Claude num **designer profissional** embutido. Em vez de você descrever no prompt "faz tipo o Stripe, mas mais escuro", a skill já carrega centenas de padrões testados e aplica automaticamente.

## Os números

- **57 UI Styles** — minimalista, glassmorphism, cyberpunk, bento, neobrutalism...
- **95 Color Palettes** — combinações já testadas, nada de cor inventada na hora
- **56 Font Pairings** — fontes que comprovadamente funcionam juntas
- **29 Landing Patterns** — estruturas de alta conversão
- **161 Reasoning Rules** — princípios aplicados automaticamente

## Os resultados de comunidade

- **42.000 downloads/mês**
- **43.000 stars** no GitHub

É uma das skills mais instaladas do ecossistema Claude Code.

## Instalação

```bash
uipro init --ai claude
```

Um comando. A partir daí, toda vez que você pedir pro Claude criar uma interface, ele aplica essas regras automaticamente.

## Quando é mais útil

- **Projetos onde você não é designer** e precisa de um visual decente sem contratar ninguém
- **MVPs** que precisam parecer profissionais em dias, não semanas
- **Dashboards internos** onde o design só precisa ser consistente e limpo

## Limitação sincera

É ótima pra ter um **ponto de partida profissional**. Pra identidade visual única (marcas com personalidade forte), ainda vale ter um designer humano refinando em cima do que a skill gera.
