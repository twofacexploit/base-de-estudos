## O que é

É uma **habilidade especial** que transforma a IA num **designer profissional embutido**. Em vez de você tentar descrever com palavras como quer que o site fique, a IA já carrega centenas de regras de design testadas e aplica automaticamente — como contratar um designer que já conhece todos os estilos e tendências.

## Os números

- **57 Estilos visuais** — moderno, escuro, colorido, minimalista, futurista e muito mais
- **95 Paletas de cores** — combinações já testadas que ficam bonitas juntas
- **56 Combinações de fontes** — letras que combinam entre si comprovadamente
- **29 Estruturas de página** — layouts que comprovadamente vendem e convertem visitantes
- **161 Regras de design** — princípios aplicados automaticamente em cada projeto

## Os resultados da comunidade

- **42.000 downloads por mês**
- **43.000 estrelas** no GitHub (forma como programadores mostram que gostam de algo)

É uma das habilidades mais populares do mundo da IA para criação de sites.

## Como instalar

```bash
uipro init --ai claude
```

Um único comando. Depois disso, toda vez que você pedir para a IA criar uma tela ou interface, ela aplica todas essas regras automaticamente.

## Quando é mais útil

- Quando você **não é designer** e precisa de um resultado bonito sem contratar ninguém
- Para criar a **primeira versão** de um produto rapidamente
- Para **painéis internos** de empresas onde o importante é ser limpo e organizado

## Uma limitação honesta

É excelente para ter um ponto de partida profissional e bonito. Mas para uma identidade visual única e muito personalizada — marca com personalidade forte — ainda vale ter um designer humano para refinar.

## Exemplos do dia a dia

1. Uma cozinheira quer criar um site para vender marmitas, mas nunca fez nada de design na vida. Com o UI UX Pro Max, ela pede para a IA criar o site e recebe algo visualmente profissional — como um kit de decoração que já vem com tudo combinando.

2. Uma pequena empresa precisa de um painel interno para acompanhar vendas. A IA cria algo limpo e organizado automaticamente, sem precisar de um designer — como preencher um formulário que já vem com o layout certo.
