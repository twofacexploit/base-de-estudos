## A ideia

Em vez de abrir dez programas diferentes para fazer uma análise de segurança, você pode instalar extensões no próprio navegador e ter tudo em um só lugar. É como transformar um carro comum em um carro de polícia adicionando equipamentos extras — o veículo é o mesmo, mas agora tem superpoderes.

## Reconhecimento

| Extensão | O que faz |
|---|---|
| **Wappalyzer** | Mostra quais tecnologias o site usa por baixo dos panos |
| **BuiltWith** | Mostra o histórico de tecnologias que o site já usou |
| **Shodan** | Mostra problemas de segurança conhecidos do site enquanto você navega |
| **WaybackMachine** | Acessa versões antigas do site e páginas que foram removidas |
| **CYFARE Reconner** | Mostra informações técnicas do domínio num painel lateral |

## Endpoints e links

| Extensão | O que faz |
|---|---|
| **EndPointer** | Descobre endereços escondidos dentro do código do site |
| **Link Gopher** | Lista todos os links da página |
| **FancyTracker** | Coleta todos os campos que o site aceita como entrada |
| **DotGit** | Detecta quando o código-fonte do site foi exposto por acidente |

## Manipulação de requests

| Extensão | O que faz |
|---|---|
| **ModHeader** | Modifica informações de identificação enviadas ao site |
| **Requestly** | Intercepta e altera dados enviados e recebidos |
| **Open Multiple URLs** | Abre uma lista de endereços de uma só vez |
| **User-Agent Switcher** | Faz o navegador se passar por celular, robô ou outro programa |

## Sessão e cookies

| Extensão | O que faz |
|---|---|
| **Cookie-Editor** | Visualiza e edita os "passes de entrada" salvos pelo site |
| **FoxyProxy** | Direciona o tráfego pelo Burp Suite com um clique |
| **CORS Everywhere** | Testa uma configuração errada comum em sites |

## Segurança e análise

| Extensão | O que faz |
|---|---|
| **SecurityHeaders.io** | Dá uma nota de A+ a F para as configurações de segurança do site |
| **HTTP Header Live** | Mostra informações técnicas do site em tempo real |
| **Retire.js** | Detecta programas desatualizados com brechas conhecidas |
| **TruffleHog** | Encontra senhas e chaves secretas expostas no código do site |

## Multipropósito

| Extensão | O que faz |
|---|---|
| **HackTools** | Coleção de ferramentas prontas para testes de segurança |
| **HackBar** | Ajuda a testar ataques comuns diretamente na barra do navegador |
| **Mitaka** | Pesquisa informações sobre endereços e sites com clique direito |
| **JS/CSS/HTML Beautify** | Organiza código bagunçado para facilitar a leitura |
| **OWASP PTK** | Lista de verificação oficial de segurança integrada ao navegador |

## Como começar

Não instale tudo de uma vez — fica confuso. Comece com apenas 4 extensões: **Wappalyzer + Cookie-Editor + FoxyProxy + HackTools**. Adicione as outras conforme for precisando.

## Exemplos do dia a dia

1. Um pesquisador de segurança usa o Wappalyzer para descobrir rapidamente que um site usa uma versão antiga de WordPress, e o Retire.js para confirmar que há plugins desatualizados com brechas conhecidas — tudo isso sem sair do navegador.

2. Um estudante que está aprendendo segurança usa o Cookie-Editor para ver e modificar os "passes de entrada" de um site de laboratório, entendendo como ataques de roubo de sessão funcionam na prática.
