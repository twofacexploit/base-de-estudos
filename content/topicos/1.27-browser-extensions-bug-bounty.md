## A ideia

Você não precisa abrir 10 ferramentas diferentes pra fazer bug bounty — dá pra ter **quase tudo dentro do navegador**, via extensões. Essas 25 cobrem o fluxo completo.

## Reconhecimento

| Extensão | O que faz |
|---|---|
| **Wappalyzer** | Detecta tecnologias do site (framework, CMS, banco) |
| **BuiltWith** | Histórico de tecnologias e mudanças ao longo do tempo |
| **Shodan** | Mostra vulnerabilidades conhecidas do host enquanto você navega |
| **WaybackMachine** | Acesso a versões antigas e endpoints removidos |
| **CYFARE Reconner** | DNS, WHOIS, SSL num painel lateral |

## Endpoints e links

| Extensão | O que faz |
|---|---|
| **EndPointer** | Extrai todos os endpoints escondidos no JavaScript |
| **Link Gopher** | Mapeia todos os links da página |
| **FancyTracker** | Coleta todos os parâmetros aceitos |
| **DotGit** | Detecta repositórios Git expostos acidentalmente (um clássico!) |

## Manipulação de requests

| Extensão | O que faz |
|---|---|
| **ModHeader** | Modifica headers HTTP (X-Forwarded-For, Host...) |
| **Requestly** | Intercepta e modifica requests e responses |
| **Open Multiple URLs** | Abre lista de subdomínios de uma só vez |
| **User-Agent Switcher** | Finge ser mobile, bot, ou browser antigo |

## Sessão e cookies

| Extensão | O que faz |
|---|---|
| **Cookie-Editor** | Visualiza e edita cookies |
| **FoxyProxy** | Gerencia proxy para Burp Suite com 1 clique |
| **CORS Everywhere** | Testa misconfiguration de CORS |

## Segurança e análise

| Extensão | O que faz |
|---|---|
| **SecurityHeaders.io** | Nota de A+ a F para headers de segurança |
| **HTTP Header Live** | Headers em tempo real |
| **Retire.js** | Detecta bibliotecas JavaScript com CVEs conhecidos |
| **TruffleHog** | Encontra API keys e secrets no JavaScript |

## Multipropósito

| Extensão | O que faz |
|---|---|
| **HackTools** | Reverse shells, payloads, encoders prontos |
| **HackBar** | Helpers de SQLi e XSS na barra do browser |
| **Mitaka** | OSINT de IPs e domínios com clique direito |
| **JS/CSS/HTML Beautify** | Formata código minificado para leitura |
| **OWASP PTK** | Checklist oficial OWASP integrado |

## Como começar

Não instale tudo de uma vez — vira bagunça. Comece com **Wappalyzer + Cookie-Editor + FoxyProxy + HackTools** (4 extensões) e vá adicionando conforme precisa.
