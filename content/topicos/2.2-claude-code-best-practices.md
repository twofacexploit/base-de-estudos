## Os 8 conceitos fundamentais

Pense no Claude Code como uma cozinha profissional. Cada conceito abaixo é um elemento importante dessa cozinha:

| Conceito | O que é |
|---|---|
| **Commands** | Atalhos rápidos — como apertar um botão que faz uma tarefa inteira (`/revisar-codigo`) |
| **Sub-Agents** | Assistentes menores e especializados dentro do assistente principal — como ter um auxiliar de cozinha |
| **Skills** | Receitas prontas que a IA pode reutilizar sempre que precisar |
| **Workflows** | Listas de tarefas que acontecem automaticamente uma após a outra |
| **Hooks** | Gatilhos automáticos — como o alarme que toca quando o bolo fica pronto |
| **MCP Servers** | Conexões com outros serviços — como o telefone que liga sua cozinha ao fornecedor |
| **Plugins** | Pacotes completos que juntam tudo acima num só lugar |
| **Settings** | As regras gerais de como a cozinha funciona |

## Como esses conceitos se encaixam

Tudo funciona em camadas, de cima para baixo. As configurações gerais definem as regras; as conexões trazem informações de fora; os assistentes especializados cuidam de partes específicas; e os atalhos e gatilhos fazem tudo rodar automaticamente.

## Por que isso importa

A maioria das pessoas usa a IA como se fosse só uma caixa de perguntas e respostas — fazem uma pergunta, recebem uma resposta. Quem aprende esses 8 conceitos transforma a IA numa verdadeira **equipe de trabalho automatizada**, capaz de fazer 10 vezes mais em menos tempo.

## O caminho recomendado

1. Use o chat básico por pelo menos uma semana
2. Crie seu primeiro atalho (**Command**) personalizado
3. Crie uma **Skill** para o tipo de trabalho que você faz mais
4. Conecte um serviço externo (**MCP Server**), como o GitHub
5. Só então experimente assistentes menores (**Sub-Agents**) e sequências automáticas (**Workflows**)

## Exemplos do dia a dia

1. Um dono de padaria configura um "Command" que, com uma só palavra, faz a IA gerar o cardápio da semana, calcular ingredientes e enviar a lista de compras — tudo de uma vez, como apertar um botão na padaria.

2. Um professor cria uma "Skill" ensinando à IA como corrigir redações no estilo dele. Dali em diante, toda vez que pedir uma correção, a IA já sabe exatamente como ele gosta — sem precisar explicar de novo.
