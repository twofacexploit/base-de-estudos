## O que é

"Code review" (revisão de código) é quando um profissional experiente olha para tudo que foi construído e aponta problemas, melhorias e perguntas importantes. Normalmente isso custa caro e leva dias. Com a IA, acontece em minutos — como ter um inspetor de imóveis que avalia cada canto da casa e te entrega um relatório completo.

## O comando que faz isso acontecer

```
You'll act as a professional code reviewer and tech lead 
for this whole codebase. Review all my project code structure, 
each file, each page, each endpoint, each logic, and write me 
a QUESTIONS.md with all the architectural, refactoring and 
technical questions you found odd or need more info about.
[...] ultrathink about this
```

## O poder do "ultrathink"

A palavra **"ultrathink"** diz para a IA **pensar muito mais profundamente** antes de responder — explorar mais possibilidades, mais ângulos, mais detalhes. É a diferença entre uma consulta rápida e uma análise completa e cuidadosa.

## Resultado real documentado

Em uma única análise, a IA gerou **68 perguntas** organizadas em **7 categorias**:

| Categoria | Quantidade |
|---|---|
| Segurança | 17 |
| Estrutura geral | 27 |
| Telas e interfaces | 16 |
| Dados | 10 |
| Testes | 4 |
| Qualidade do código | 7 |
| Desempenho | 5 |

Tudo salvo num arquivo chamado `QUESTIONS.md` — algo que levaria dias para um profissional humano produzir.

## Como usar na prática

```
Comando → IA gera lista de perguntas → você responde cada uma
        → IA implementa as correções → você revisa e repete
```

Você decide **o quê** precisa ser feito. A IA faz **o como**. É uma parceria: humano pensa, máquina executa.

## Por que isso importa

Uma revisão profissional de projeto custa caro — horas e horas de um especialista. Esse método entrega **grande parte do valor em minutos**, praticamente de graça.

## Exemplos do dia a dia

1. Um empreendedor lançou seu site de vendas e quer saber se tem problemas escondidos antes de investir em marketing. Ele roda esse comando e recebe uma lista completa de pontos a melhorar — como uma vistoria de imóvel antes de comprar.

2. Um estudante terminou seu primeiro projeto e quer aprender o que poderia ter feito melhor. A IA aponta tudo que um professor experiente apontaria, mas de forma detalhada e instantânea — como ter um mentor particular disponível na hora.
