## O que é

Imagina plantar mil sementes de uma vez, e cada uma crescer e trazer frutos por anos. O **SEO programático** faz isso com páginas de internet: você cria **milhares de páginas automaticamente**, cada uma otimizada para aparecer quando alguém pesquisa algo específico no Google. E cada página continua trazendo visitantes mesmo enquanto você dorme.

## O fluxo com IA

```
Palavra-chave base: "melhor sistema para [tipo de negócio]"
      ↓
Variações: dentistas, advogados, imobiliárias,
           restaurantes, barbearias... ×10.000
      ↓
Claude cria um texto único e útil para cada nicho
      ↓
Firecrawl coleta dados reais do setor para enriquecer cada página
      ↓
Resultado: 10.000 páginas, cada uma aparecendo para uma busca diferente
```

## A matemática

```
10.000 páginas × 30 visitas/mês = 300.000 visitantes  
2% de conversão                  = 6.000 clientes  
R$50/mês por cliente             = R$300.000/mês  
```

Uma vez construída, essa máquina traz visitantes sem você precisar pagar por anúncios todos os meses.

## A regra de ouro

O Google penaliza páginas sem valor real — textos copiados ou sem substância ficam invisíveis nas buscas. O segredo é **quantidade com qualidade**: cada página precisa realmente ajudar quem a encontrou.

Cada página precisa:
- Responder a uma dúvida real de quem pesquisou
- Trazer informações específicas sobre aquele nicho
- Ser fácil de ler e navegar
- Parecer feita por alguém que entende do assunto

## Quem já faz isso

Empresas gigantes que você conhece:

- **Booking.com** — uma página para cada combinação de "hotel em [cidade]"
- **Airbnb** — páginas automáticas para cada tipo de acomodação em cada lugar
- **Glassdoor** — uma página para cada empresa + cargo + cidade
- **Yelp** — página para cada restaurante + categoria + localização

Essas empresas construíram o SEO programático antes da IA existir. Agora, com Claude + Firecrawl, **qualquer pessoa** pode fazer o mesmo em semanas — não em anos.

## O trade-off ético

Essa técnica pode ser usada para entregar conteúdo genuinamente útil em escala — ou para inundar a internet com páginas vazias só para ganhar cliques. O Google vai identificando quem cruza essa linha e remove essas páginas das buscas. A aposta certa sempre foi na qualidade.

## Exemplos do dia a dia

1. Um advogado especializado em acidentes de trânsito cria uma página diferente para "advogado de acidente de trânsito em [nome da cidade]" para cada cidade do Brasil. Cada página tem informações específicas sobre aquela cidade. Resultado: o site aparece no Google para qualquer pessoa que pesquise isso em qualquer cidade — sem pagar anúncios.

2. Uma plataforma de cursos online cria uma página para "curso de [profissão] para [tipo de pessoa]" — "curso de Excel para iniciantes", "curso de Excel para secretárias", "curso de Excel para contadores". Cada página é diferente e responde perguntas específicas daquele público.
