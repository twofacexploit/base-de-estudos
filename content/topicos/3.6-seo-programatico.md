## O que é

**SEO programático** = criar **milhares de páginas únicas** automaticamente, cada uma otimizada para uma keyword específica. Cada página ranqueia para uma busca, e o tráfego orgânico se multiplica.

## O fluxo com IA

```
Keyword base: "best CRM for [nicho]"
      ↓
Variações: dentistas, advogados, imobiliárias,
           restaurantes, barbearias... ×10.000
      ↓
Claude gera conteúdo único e relevante para cada nicho
      ↓
Firecrawl coleta dados reais do setor para enriquecer
      ↓
Resultado: 10.000 páginas, cada uma ranqueando pra uma variação
```

## A matemática

```
10.000 páginas × 30 visitas/mês = 300.000 visitantes  
2% de conversão                  = 6.000 clientes  
$10/mês por cliente              = $60.000/mês MRR  
```

É uma **máquina de aquisição** que, uma vez construída, gera tráfego mesmo enquanto você dorme.

## A regra de ouro

**Conteúdo genuinamente útil.** O Google penaliza spam — páginas geradas sem substância são invisibilizadas. O segredo não é **quantidade**, é **qualidade multiplicada**.

Cada página precisa:
- Responder a uma pergunta real
- Trazer **dados específicos** daquele nicho
- Ter estrutura clara e navegável
- Não parecer "copy-paste com trocar palavra"

## Quem já faz isso

Empresas gigantes que você conhece:

- **Booking.com** — uma página pra cada combinação de "hotel em [cidade]"
- **Airbnb** — páginas programáticas pra cada tipo de acomodação
- **Glassdoor** — uma página pra cada empresa + vaga + cidade
- **Yelp** — página pra cada restaurante + categoria + localização

Todos construíram SEO programático **antes** de IA existir. Agora, com Claude + Firecrawl, **qualquer pessoa** pode construir isso em semanas — não em anos.

## O trade-off ético

Essa técnica pode ser usada pra **bem** (entregar conteúdo útil em escala) ou pra **mal** (inundar a web com spam de afiliado). A linha é fina — o Google vai caçando quem cruza ela.
