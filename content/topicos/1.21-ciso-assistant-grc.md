## O que é GRC

Pensa no **GRC** como o caderno de regras de uma escola: alguém precisa anotar quais regras existem, verificar se todos estão seguindo e provar para a diretoria que está tudo em ordem. Na empresa, esse "caderno de regras" é sobre segurança de dados, riscos e leis que a empresa precisa obedecer.

## O que substitui

Existem programas famosos para isso que custam entre **R$ 250.000 e R$ 1.000.000 por ano**. O CISO Assistant faz grande parte do mesmo trabalho de graça — como se fosse a versão gratuita de um software que normalmente custa uma fortuna.

## Frameworks suportados

- **ISO 27001** — conjunto de regras internacionais de segurança da informação
- **NIST** — padrão de segurança usado pelo governo americano
- **LGPD** — lei brasileira que protege os dados pessoais das pessoas
- **GDPR** — versão europeia da LGPD
- **PCI-DSS** — obrigatório para quem aceita pagamento por cartão de crédito
- **SOC 2** — auditoria de segurança para empresas de software

## Para quem importa

- Startups brasileiras que precisam provar que seguem a LGPD
- Empresas que querem vender para clientes nos Estados Unidos
- Qualquer negócio que aceita cartão de crédito
- Quem quer seguir a ISO 27001 sem pagar consultoria cara

## A realidade

Esse tipo de documentação é chato, mas é o que permite fechar contratos grandes. Uma empresa bem organizada nessa área consegue fechar parcerias que outra empresa desorganizada jamais conseguiria.

## Exemplos do dia a dia

1. Uma startup de saúde digital usa o CISO Assistant para organizar toda a documentação que prova que ela segue a LGPD — como montar um dossiê para mostrar para o governo que a empresa cuida dos dados dos pacientes direitinho.

2. Uma pequena empresa de software que quer vender para uma grande rede de supermercados usa a ferramenta para se preparar para a auditoria de segurança que o cliente exige, sem precisar contratar uma consultoria caríssima.
