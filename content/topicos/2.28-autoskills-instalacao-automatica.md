## Como usar

```bash
npx autoskills             # detecta e instala tudo
npx autoskills --dry-run   # preview, sem instalar
```

## O que ele faz — 4 passos

1. **Lê** `package.json`, `requirements.txt`, `go.mod` e outros arquivos de config
2. **Identifica** as tecnologias que seu projeto usa
3. **Busca** no registry de skills aquelas que fazem sentido pra essas tecnologias
4. **Instala** em `.claude/skills/` automaticamente

## 50+ tecnologias suportadas

React, Next.js, Vue, Svelte, Astro, Tailwind, Supabase, Neon, Playwright, Expo, Stripe, Prisma, Cloudflare, AWS, Vercel, GSAP, Bun, Deno, Hono, NestJS, Spring Boot — e mais.

Praticamente qualquer stack moderna é detectada.

## Por que importa

Antes: cada vez que você começava um projeto novo, tinha que **buscar manualmente** quais skills fazem sentido. Pra uma stack complexa (Next + Supabase + Stripe + Vercel + Playwright), isso levava **horas**.

Agora: **um comando**, tudo configurado. Você sai do zero para um ambiente Claude Code otimizado em **menos de um minuto**.

## O modo `--dry-run`

Antes de instalar, veja o preview do que vai ser feito. Ideal pra:

- Auditar o que está sendo adicionado ao seu projeto
- Aprender sobre skills que você não conhecia
- Evitar instalações indesejadas

## A filosofia por trás

Automação **inteligente** é melhor que automação **cega**. AutoSkills não instala tudo que existe — instala o que é **relevante para o seu contexto específico**. Isso é o tipo de ferramenta que define o futuro do tooling de IA.
